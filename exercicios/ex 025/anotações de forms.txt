inputs:
submit - submeter
reset - limpar
Number - numéricos
Month - mês
date- data
time - tempo, hora
radio - item para marcar
checkbox - caixa de checagem
color - cor
range - nivel
file - arquivo

Outputs:
output - oniput

tags: 
fieldset - cria uma caixa contendo todos os campos
legend - acrescenta na legenda do formulário  
select - Botão de seleção
option - Opção do Botão de seleção
optgroup - grupo de opções
datalist - lista de dados
textarea - Area de texto



Parâmetros:
required - Requerido, obrigatório
minlength - tamanho mínimo de letras
maxlength - tamanho máximo de letras
size - tamanho
placeholder - colocar coisas dentro da caixa
autocomplete - autocompleta 
	ex: username - nome de usuários, current-password - senhas salvas
step - passa de n em n (n - número definido por vc)
min - valor mínimo que pode ser inserido (numérico) 
max - valor máximo que pode ser inserido (numérico)
Value - valor que preenche previamente
pattern -> faz a validacao do campo 
selected - seleciona um option
for - para onde
list - implementa a lista de sugestao
